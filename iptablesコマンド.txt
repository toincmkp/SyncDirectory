・iptablesの初期化
# iptables --flush

・iptablesの設定を確認
iptables -L

・iptablesの設定を保存
service iptables save

・ポート80で受信したTCPの受信パケットを破棄
iptables -A INPUT -p tcp --dport 80 -j DROP

・ICMPのパケットをすべて破棄
iptables -A INPUT -p icmp -j DROP

・192.168.1.4以外のICMPパケットをすべて破棄
iptables -A INPUT -s 192.168.1.4 -p icmp -j ACCEPT
iptables -A INPUT -p icmp -j DROP

・MACアドレスによるフィルタリング
iptables -A INPUT -m mac --mac-source 00:08:74:14:CB:10 -j ACCEPT
